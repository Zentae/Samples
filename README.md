# AccountHandler

### Description
*Ce plugin permet une gestion des comptes, pour un serveur mini-jeux. Il gère les comptes depuis une ```BDD (MySQL)``` et ```Redis```*

### Plugins

*Ce plugin nécessite les libraires suivantes*

| Plugin/Lib | VERSION |
| ------ | ------ |
| Bungeecord | 1.8 - 1.14.4  |
